# Task Management System

## Project Overview

The Task Management System is a web application that helps users organize and manage their daily tasks efficiently. Users can register, log in, create tasks, update tasks, delete tasks, and track task completion.


## Project Objective

To provide a simple and user-friendly platform where users can manage their daily activities and improve productivity through task tracking.


## User Roles

### User

- Register an account
- Login securely
- Logout from the system
- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- View all tasks on the dashboard

## Functional Requirements

### Authentication

- User Registration
- User Login
- User Logout

### Task Management

- Add Task
- Edit Task
- Delete Task
- View Tasks
- Mark Tasks as Completed

### Dashboard

- Display all user tasks
- Show completed and pending tasks
- Provide task management options
