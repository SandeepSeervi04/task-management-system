# Task Management System

## Project Overview

The Task Management System is a web application that helps users organize and manage their daily tasks efficiently.


## Project Objective

To provide users with a simple platform where they can create, update, delete, and track their tasks.


## User Roles

### User

- Register an account
- Login to the system
- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed


## Functional Requirements

### Authentication

- User Registration
- User Login
- User Logout

### Task Management

- Add Task
- Edit Task
- Delete Task
- View Tasks
- Complete Tasks


## Technology Stack

### Frontend

- HTML
- CSS
- JavaScript

### Backend

- Node.js
- Express.js

### Database

- MongoDB

### Authentication

- JWT


## Expected Outcome

A fully functional web application where users can manage their daily tasks securely.
