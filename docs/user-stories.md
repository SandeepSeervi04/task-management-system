# User Stories

## Registration

As a user, I want to register an account so that I can access the task management system.


## Login

As a user, I want to log in securely using my email and password so that my tasks remain private.


## Dashboard Access

As a user, I want to access a dashboard so that I can view and manage all my tasks in one place.


## Create Task

As a user, I want to create new tasks so that I can organize my daily activities.


## Edit Task

As a user, I want to edit existing tasks so that I can update information when needed.


## Delete Task

As a user, I want to delete tasks that are no longer required.


## Complete Task

As a user, I want to mark tasks as completed so that I can track my progress.


## Responsive Design

As a user, I want the application to work on both desktop and mobile devices so that I can manage tasks from anywhere.


## Logout

As a user, I want to log out securely when I finish using the application.
