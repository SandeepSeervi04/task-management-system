# User Stories

## Registration

As a user, I want to register an account so that I can access the system.

## Login

As a user, I want to login securely using my email and password.

## Create Task

As a user, I want to create new tasks so that I can organize my work.

## Edit Task

As a user, I want to edit tasks when information changes.

## Delete Task

As a user, I want to remove tasks that are no longer needed.

## Complete Task

As a user, I want to mark tasks as completed after finishing them.
